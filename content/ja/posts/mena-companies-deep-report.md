# 中東インターネットユニコーン深層レポート：6社の巨額評価額を支えるICT戦略

**著者：Tech Observer | 2025年11月**

> 本レポートは、中東・北アフリカ（MENA）地域を代表する6社のインターネットユニコーン企業——Careem、Noon、Shahid、Souq/Amazon、Talabat、Yalla——を対象に、財務実績、ICT投資、クラウドサービスアーキテクチャ、データセンター戦略、最新の戦略動向を深く分析する。これら6社の評価額合計は150億ドルを超え、配車、Eコマース、動画ストリーミング、フードデリバリー、ソーシャルエンターテインメントなど多岐にわたる事業を展開している。

---

##  1、Careem：Uber系スーパーアプリのクラウドファースト戦略

**評価額：31億ドル（Uber買収額） | 本社：ドバイ | 展開地域：15カ国120都市**

### 主な発見

- **スーパーアプリ戦略**：2021年に始動し、配車、フードデリバリー、決済（Careem Pay）、即時配送を統合
- **e&からの戦略的投資**：2023年にUAEテレコムグループから4億ドルの資金調達を実施し、スーパーアプリ事業の拡大を加速
- **マルチクラウドアーキテクチャ**：800以上のマイクロサービスを稼働させ、主にAWSとGCPを基盤として構成
- **AI駆動**：不正検知（AWS Neptuneグラフデータベース）、需要予測、パーソナライズドレコメンデーション
- **クラウドファースト戦略**：自社データセンターは構築せず、AWS/GCPの中東（バーレーン、UAE）インフラを最大限活用
- **2025年の収益は28億ドル超を予測**

### 技術アーキテクチャ

```
AWS (コア) + GCP (ビッグデータ分析)
└── 800以上のマイクロサービス
    ├── AI/ML (GCP BigQuery)
    ├── 決済 (Careem Pay)
    └── 物流追跡
```

##  2、Noon：10億ドル帝国のAzureクラウドシフト

**評価額：10億ドル超 | 本社：ドバイ | 展開国：UAE、サウジアラビア、エジプト**

### 主な発見

- **創業背景**：Emaar創業者アルアバール氏がサウジアラビア公共投資基金（PIF）と提携し、2016年に10億ドルの初期投資を調達
- **事業エコシステム**：Noon.com + Noon Food + Noon Minutes（15分即配）+ Noon Pay
- **クラウドサービス**：Noon Paymentsのコア業務はすでにMicrosoft Azureへ移行済み
- **2025年の流通取引総額（GMV）は50～70億ドル規模に達し、黒字化が目前**
- **デュアル上場計画**：今後18～24カ月以内にサウジアラビアとUAEでのIPOを予定
- **自動運転配送**：2025年11月にアブダビで実証実験を開始

### 財務ハイライト

| 指標 | データ |
|------|------|
| 年間GMV | 50～70億ドル |
| 成長率 | 年平均30%～50% |
| 対象市場 | UAE、サウジアラビア、エジプト |
| ICT投資比率 | 運営支出の15%～25% |

##  3、Shahid：MBCグループのストリーミング野望を支えるクラウド

**親会社：MBC Group（サウジPIFが支配権を保有）| 本社：ドバイ | 展開地域：アラブ22カ国**

### 主な発見

- **市場での地位**：MENA地域最大のアラビア語VODプラットフォーム
- **コンテンツ面の優位性**：MBCグループの全テレビ放送コンテンツの独占配信権を保有し、膨大なアラビア語オリジナルコンテンツを誇る
- **サウジアラビアでの位置付け**：サウジ「ビジョン2030」のデジタルエンターテインメント戦略の中核として、PIFから強力な支援を受ける
- **マルチクラウド戦略**：主にAWSとCDNを利用し、大規模な動画ストリーミングを支える
- **ユーザー規模**：有料会員数は継続的に増加し、サウジアラビアのユーザー比率が40%を超える

##  4、Souq → Amazon.ae：アマゾンの中東戦略

**親会社：Amazon（NASDAQ: AMZN）| 買収額：5.75億ドル（2017年）**

### 主な発見

- **歴史的ポジション**：Souq.comは「中東のアマゾン」と称され、2005年に創立された中東で最も古いEコマースプラットフォームの一つ
- **アマゾンによる買収**：2017年に5.75億ドルで買収され、その後Amazon.ae/.sa/.egへとブランド刷新
- **技術統合**：全面的にAWSへ移行し、アマゾンの技術スタックを活用して物流、決済、レコメンドシステムを強化
- **FBA（フルフィルメントby Amazon）**：アマゾンの成熟した物流システムを導入し、配送体験を大幅に向上
- **AWS中東インフラ**：Amazon.aeはAWS中東（バーレーン）リージョンに深く依存

### 技術的優位性

```
アマゾンの技術によるエンパワーメント：
├── AWS (クラウドインフラ)
├── DynamoDB (データベース)
├── SageMaker (AIレコメンド)
└── 物流ネットワーク（FBA）
```

##  5、Talabat：湾岸フードデリバリー戦争の技術基盤

**親会社：Delivery Hero（フランクフルト上場）| 本社：クウェート | 展開国：6カ国**

### 主な発見

- **Delivery Hero子会社**：2020年に欧州のフードデリバリー大手Delivery Heroに買収される
- **技術スタック**：主にAWS上にマイクロサービスアーキテクチャを構築し、高同時接続の注文処理をサポート
- **市場カバレッジ**：クウェート、サウジアラビア、UAE、カタール、バーレーン、オマーン
- **配達員ネットワーク**：湾岸6カ国で10万人以上の配達員を運用
- **急拡大**：M&Aを通じて地場のフードデリバリープラットフォームを統合し、湾岸地域で独占的地位を形成

##  6、Yalla：高収益を支えるテクノロジー

**NYSE: YALA | 本社：ドバイ | 2020年上場（中東初の米国上場ユニコーン）**

### 主な発見

- **独自のビジネスモデル**：音声SNS（Yalla）+ ゲームSNS（Yalla Ludo）で、アラブ市場に特化
- **驚異的な収益性**：純利益率は常に40%超を維持し、2024年の売上高は3億ドルを突破
- **軽資産型テクノロジー**：主にAWSを基盤に構築、重厚な物流インフラは不要
- **ローカライゼーションの深さ**：製品は完全にアラブ人ユーザー向けに設計され、アラビア語NLPに対応
- **2024年の純利益は1.2億ドル超**

### 財務実績

| 年 | 売上高 | 純利益 | 純利益率 |
|------|------|--------|--------|
| 2022 | 2.4億ドル | 0.9億ドル | 37.5% |
| 2023 | 2.8億ドル | 1.1億ドル | 39.3% |
| 2024 | 3.1億ドル | 1.25億ドル | 40.3% |

##  7、MENAインターネット企業のICT共通点分析

### クラウドサービスプロバイダー分布

| クラウドサービスプロバイダー | 主要顧客 | データセンター所在地 |
|----------|----------|--------------|
| AWS | Careem, Talabat, Yalla, Amazon.ae | バーレーン（中東） |
| Microsoft Azure | Noon (Payments), Shahid | UAE |
| Google Cloud | Careem (BigQuery) | UAE |

### 主要トレンド

1. **マルチクラウド戦略**：主要企業は、単一ベンダー依存を避けるため、2～3社のクラウドサービスを併用
2. **クラウドファースト（Cloud-First）**：Yallaを除き、主要企業は自社データセンターを保有しない選択をしている
3. **AI駆動**：全企業がレコメンド、不正検知、需要予測などの領域でAIを深く活用
4. **海外進出と中国**：Noonは中国市場へ進出し、TalabatやYallaは湾岸地域のローカライゼーションに注力

MENAのインターネット市場は、「市場原理よりも政策主導の色彩が強い」特殊な存在である。サウジPIFの強力な布陣により、ShahidとNoonは他国では想像もできない資本支援を受けている。一方、Yallaの高収益は、ローカライゼーションを極限まで追求すれば、資金調達に頼らなくとも十分にやっていけることを証明している。

## Key Takeaways

- **市場は政策主導**：サウジPIFが最大の資金源であり、ShahidとNoonの台頭は、本質的にはソブリン・ウェルス・ファンドによる戦略的布陣であり、純粋な市場原理に基づく行動ではない
- **AWSがMENAクラウド市場を支配**：主要企業（Careem、Talabat、Yalla、Amazon.ae）のコアアーキテクチャはいずれもAWSを基盤としており、中国系クラウドベンダーの参入機会は限定的
- **スーパーアプリ化が共通の最終形態**：Careemは配車＋デリバリー＋決済を統合し、NoonはEコマース＋決済＋即時配送のエコシステムを構築。プラットフォーム化がMENAインターネットの進化の方向性である
- **Yallaの収益モデルは注目に値する**：アラブ市場へのローカライズに特化し、純利益率40%超を達成。MENAインターネット業界では稀な「自己造血」事例
- **中国企業の海外進出のチャンスはゲームと新エネルギー分野**：純粋なインターネットプラットフォームはローカライゼーションの障壁が高く、ゲーム（miHoYo）やEコマース（SHEIN）が成功モデルとなっている
