## 核心的見解

> タイのIDC市場は現在、「新興」から「成熟」への移行という重要な岐路に立っています。2025年の市場規模は約**18.9億ドル**で、2030年には**49億ドル**に達し、年平均成長率（CAGR）は約17%と予測されています。AIコンピューティング需要の急増、中国テクノロジー企業の大規模な海外進出、ハイパースケールクラウド事業者の現地展開という3つの力が重なり合い、タイを東南アジアのデータセンター版図の中核へと押し上げています。

##  1、市場規模と主要指標

![バンコクの夜景](https://images.unsplash.com/photo-1509343256512-d77a5cb3791b?w=800&q=80)
*バンコクはタイのデータセンターの絶対的中心地であり、既存容量の約75%を占める*

### 主要データ

| 指標 | データ |
|------|------|
| 2025年(推定)市場規模 | 18.9億ドル |
| 2030年(予測)市場規模 | 49億ドル |
| 年平均成長率（CAGR） | 約17% |
| 既存の総容量（2024年） | 約90 MW |
| 2026年(予測)容量 | 約200～300 MW |
| バンコクのシェア | 約75% |
| 東南アジアランキング（容量） | 第4位（シンガポール、インドネシア、マレーシアに次ぐ） |

### 東南アジア5カ国との横断比較

タイは東南アジアで4位にランクされており、既存容量は約90MWで、シンガポール（約980MW）、インドネシア（約400～500MW）、マレーシア（約350MW）を下回っています。しかし、タイの成長ドライバーは質的に変化しています。AIハイパースケールプロジェクトが集中的に着地することで、2026～2027年にかけて容量は飛躍的な成長を遂げると見られています。シンガポールと比較すると、タイは**低コストの電力**（商業用電気料金は約THB 4.20/kWh、約0.11ドル）、**緩和的な政策環境**（BOI免税、EEC経済回廊）により、明確な差別化競争優位性を形成しています。

##  2、競争構造: 地場大手が主導し、中国勢が急速に浸透

### 地場系および国際系事業者

| 事業者 | タイプ | 既存容量 | コア競争力 |
|--------|------|----------|----------|
| **True IDC** | 地場系（CPグループ） | 150 MW以上 | タイNo.1、AIハイパースケール、マイクロソフトの戦略的パートナー |
| **STT GDC Thailand** | 国際系（シンガポール） | 約30～50 MW | シンガポールのSTT GDCの裏付け、ESG優先 |
| **AIS / Gulf** | 地場系連合 | 計画中 | 政府支援、マイクロソフトとの共同建設計画 |

True IDCは、タイのデータセンター市場において議論の余地なく第一位の事業者です。23年の運営実績を持ち、既存の電力容量は150MWを超え、タイで初めて高度なGPU処理システムをホスティングできるデータセンタープロバイダーです。親会社であるCPグループ（2025年の売上高820億ドル超）は、比類のない法人顧客ネットワークと資本力の裏付けを提供しています。

### 建設中・計画中の主要プロジェクト

**True IDCの2大メガプロジェクト:**

- **EECハイパースケールデータセンター**: 東部経済回廊（EEC）で既に正式着工、計画容量は250MW、総投資額は770億バーツ（約23.6億ドル）で、BOIの承認を取得。第1期は**2027年稼働**予定で、完全なモジュラーアーキテクチャを採用。
- **ラヨーン団地**: 2025年11月に5.6億ドルのシンジケートローンによる資金調達を実施、計画容量は102.6MWで、**2027年の稼働**を目指す。

両プロジェクトによる新規容量は合計約350MW、総投資額は30億ドル以上となり、True IDCを東南アジア最大級のデータセンター事業者の仲間入りさせることになります。

戦略的資本面では、ブラックロック傘下のグローバル・インフラストラクチャー・パートナーズ（GIP）がCPグループおよびTrue IDCとの協業に合意し、今後3～5年で**10億ドル**を超える資本投下を実現し、事業を東南アジア全域に拡大することを目指しています。

##  3、中国テクノロジー企業の海外進出: タイの新たな成長極

これは従来の分析枠組みでは最も見落とされがちでありながら、実際の影響が最も深遠な変数です。

### インターネット大手

複数の中華系インターネット企業が、コンテンツエコシステム、データセンターインフラ、デジタルエコノミー展開を網羅する大規模な投資計画をタイで発表しています。タイは中華系インターネット企業にとって、東南アジアで最も重要な戦略的布石の一つになりつつあります。

### GDSインターナショナル（DayOneブランド）

タイBOIは、GDSインターナショナル（DayOne）が**チョンブリー県アマタ工業団地**に10億ドルを投資し、超大規模データセンターキャンパスを建設する計画を承認しました。第1期は**2026年の稼働**を予定しており、全体計画容量は100MWを超えます。2025年第2四半期には既に主要顧客との契約を完了しており、現時点で最も実質的な中国IDC海外展開の進展です。

### 秦淮データ（Bridge Data Centres）

2021年にタイ・バンコクのデータセンター買収を完了、IT容量は約5MWで、中国のサードパーティIDC企業によるタイ市場への初の布石となりました。2025年、Bridge DCは東南アジア地域拡大のため28億ドルのシニア担保付ローンを調達し、タイはその重点拡大市場の一つです。

### Alibaba Cloud（アリババクラウド）

Alibaba Cloudは、タイに最も早く進出した中国系クラウド事業者の一つで、2022年に現地初のデータセンターを設立し、**2025年2月に2番目のデータセンターを開設**しました。これによりタイでデュアルアベイラビリティゾーン構成を実現し、正式にエンタープライズクラスのクラウドサービス能力を備えました。

##  4、AIコンピューティング需要: 次段階の中核的駆動力

### BIS輸出規制が生み出す計算資源の空白

米国商務省産業安全保障局（BIS）による輸出規制は2022年以降段階的に強化され、A100、H100、H800を全面的に封鎖し、2025年にはさらにH20、L20を管理対象に加えました。これにより、タイ（および東南アジア全域）の中国インターネット企業は、**同等のNvidia GPUを合法的に購入または使用することができなくなり**、大きな計算資源の供給不足が生じています。この規制による空白が、東南アジアにおける代替計算ソリューションの導入を加速させています。

### GPUクラスター導入データ

- タイのGPUクラスター導入プロジェクト数は前年比**200%**以上増加（2024-2025年）
- True IDCはタイで**初めて高度なGPU処理システムをホスティングできる**データセンタープロバイダー
- AIハイパースケール建設コスト：タイの標準ハイパースケールは約7,000～8,000ドル/kW、AI最適化（液冷＋高密度）は約15,000～20,000ドル/kW

##  5、主要な投資機会

### 1. AIコンピューティングホスティング（確度高）

LLMのトレーニングと推論の需要急増に半導体規制のレッドラインが重なり、タイはアジア太平洋地域のAIトレーニングハブ候補地になりつつあります。GPUクラスター導入プロジェクト数は前年比200％超で増加しており、True IDCが既に先陣を切っています。

### 2. ハイパースケール建設のウィンドウ（確度高）

True IDCの2大プロジェクト計350MW、30億ドル超の総投資、DayOneチョンブリーキャンパス100MWは、2026～2027年にかけて集中的な建設ピークを形成し、機器サプライヤー（サーバー、ネットワーク、冷却）にとって高密度な調達ウィンドウとなります。

### 3. 中国インターネット企業のローカライズ（確度高）

TikTok、Shopee、Alibaba Cloudなどの企業が継続的に投資を拡大しており、ローカライズされたコンテンツ配信やコンプライアンス対応データストレージの需要が、CDNノードやエッジデータセンターの建設を促進しています。

### 4. グリーンデータセンター（長期的テーマ）

太陽光＋PPAモデルが徐々に成熟しており、タイの年間平均日照時間2,500時間は自家発電に独自の優位性をもたらします。EEC地域のBOI政策もグリーンインフラを同時に支援しています。
